Augmented Reality API(visageAR.js)

VisageAR is a simple, high-level API for augmented reality applications, allowing to put virtual object (such as eyeglasses etc.) on the user's head in real time. VisageAR API is built upon the core visageSDK API, and provided as sample implementation with full source code.

Main classes

  • VisageAR: Provides high-level AR functions - loading objects to be overlaid on the face, starting/stopping the tracking.
